Migration Guide from v1 to v2

Transitioning from /api/v1/old-feature to /api/v2/new-feature

Request changes:

    GET /api/v1/old-feature?param=value

    GET /api/v2/new-feature?new_param=new_value

Response changes:

Previous response:

{
    "old_key": "old_value"
}

Is now:

{
    "new_key": "new_value"
}

Handling Deprecated Endpoints

The deprecated endpoints will remain functional until 2023-01-01, allowing time for transition. Contact support for assistance.
